# ZeDART_ADSBDevice
Script pour créer le device ADS-B pour ZeDART

## INSTALLATION
- Installez Raspberry Pi OS en version "Raspberry Pi OS Lite" sur votre Raspberry Pi (2B minimum, mais 3 conseillé pour éviter d'avoir à ajouter une clé Wifi) à l'aide de https://www.raspberrypi.com/software/
- Veillez à rentrer les paramètres Wifi pour qu'il soit connecté à internet le temps de l'installation 
- Une fois l'OS opérationnel, tapez `curl -fsSL https://raw.githubusercontent.com/zesinger/ZeDART_ADSBDevice/main/install-zedart.sh | sudo bash`
C'est tout...
